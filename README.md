## Install

```
git clone git@github.com:yao-dev/marvel-app.git
cd marvel-app
```

## Start app

Enter in project then run :

**Dev mode**
```
npm run dev
```

**Prod mode**

```
npm run build
npm run start
```

## Unit tests & coverage

```
npm test
```

**With coverage**
```
npm run test:coverage
```
And go to (path/of/your/system/marvel-app/coverage/lcov-report/index.html)

## Deploy with now.sh

```
npm run deploy
```
